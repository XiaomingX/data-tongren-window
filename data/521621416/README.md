# 海滩的青春岁月

我无法停止对他的注视，他正在随着格雷戈里讲述的故事笑，双手在空中比划。两人都光着上身，背景是美丽的海滩。我坐在一个原木上，头发被海风轻轻吹动，享受这一刻的美好。

夕阳已经降下，今天的时间过得飞快，本可以只花两小时就到达的海滩，偏偏我们一路停停打打，嘻嘻哈哈地聊着。阿波罗、马尔科和尤西正在沙滩上用我们在某个休息站买的球玩耍，像几个无忧无虑的孩子。丹尼则在海边慢腾腾地走着，享受那份宁静与独处的时光。

“景色真美，是不是？”我旁边的萨米突然开口，打断了我的沉思。

我微笑着点头，“是啊，这趟旅行真值得。”

她递给我一个金属杯，“来，喝一口？”

我接过杯子，喝了一小口，烈酒的味道瞬间烤焦我的喉咙。“威士忌？”我将杯子递回去，看她毫不在意地喝着。

“跟这些家伙在一起太久，我的坏习惯越来越多了。”她有些无奈地笑道。

“你就没有女朋友吗？”我问。

“没有，一直都是他们。”她的目光转向格雷戈里和阿瑞斯，最后落在阿波罗身上，“不过，我跟他们相处得很好。”

“从小就认识他们，一定有不少有趣的故事吧？”我好奇地问。

萨米轻声一笑，“我当然知道一堆丢脸的事，不过克劳迪亚更厉害，她知道的更多。”

我用怀疑的眼神看着她。她似乎读懂了我的心思，举手示意，“不，我不知道克劳迪亚、阿瑞斯和阿波罗之间到底发生了什么。”

“阿波罗？”我眉头一皱。

她立刻变得语无伦次，“呃，我是说……不是说有事，只是我以为……算了，忘了吧。”

我的心思回到那次在医院，阿尔忒弥斯打了阿波罗，和我生日那天阿波罗又打了阿尔忒弥斯。目光落在丹尼身上，保护她的本能战胜了好奇心。“阿波罗跟克劳迪亚有什么关系吗？”

萨米没有回答。

我抓住她的手腕，逼她说出真相：“萨米，我不喜欢逼人，但丹尼是我最好的朋友，我愿意为她做任何事。我需要知道，我是否该告诉她放弃阿波罗。”

“如果我知道，就会告诉你，真的，瑞秋。但我一无所知。阿尔忒弥斯就像一块冰，阿波罗又太过正直，不会谈论女孩，而阿瑞斯嘛，对自己的兄弟都不那么诚实。他们之间的忠诚感令人难以捉摸。”

我相信她。想起我之前试图从阿瑞斯那里获取信息的经历，那次用情欲来换信息的计划简直是个笑话，结果一头雾水。

“热带美女们！”格雷戈里突然出现在我们面前，五官精致的小脸上挂着阳光明媚的笑容。

我忍不住笑了，就是这个家伙，精力充沛得令人发指。“你怎么总是这么有活力？”

“这就是年轻的力量。”格雷戈里坐下，目光好奇地在我们之间游走，“你们刚才在聊什么？看你们表情那么严肃？”

“没什么，开玩笑而已，”我故作轻松，抚摸着格雷戈里的头，像是在逗小狗一样。

“你简直让他不成熟。”萨米翻了个白眼，格雷戈里则用无辜的眼神回望。

“我不想给你抚摸的机会。”格雷戈里打了个小赌气，结果还是引来大家的一阵嘲笑。

夜幕降临，该准备篝火了。“应该在太阳完全消失之前点个篝火。”我突然提议。

“电影里的篝火从来都不难。”我们开始寻找木材，现实中的情况却让我们汗流浃背，毫无进展。经过八次搜索后，终于，火堆点燃了，木材劈劈啪啪地燃烧，大伙围着火堆坐下，脸上挂满笑容。

我靠在阿瑞斯的肩膀上，眼神凝视着翻腾的火焰，心中一片宁静。“我会想念你们的。”格雷戈里打破沉默，语气中透着伤感。

阿波罗丢了一根木头进去，火焰更旺。他侧目看着格雷戈里，“你至少要上大学，格雷戈里。我还得留在高中。”

丹尼看着阿波罗，目光中有一丝情愫，我忍不住想，她会不会像我每次看阿瑞斯那样，心中满是甜蜜和苦涩？

“我带了棉花糖回来！”马尔科从汽车中返回，手中提着两袋美味的甜食。

“太好了，我恨不得立刻就吃到。”萨米迫不及待地回应。

“当然，马尔科可以给你一些特别的甜点，你知道的……”格雷戈里偷讽。

“你真坏。”萨米皱起了眉。

丹尼突然插嘴，“而且，那不算甜。”

“哦——”大家齐声起哄，丹尼意识到自己说了什么，脸上瞬间罩上红晕。

这是典型的“自杀式发言”，她这一辈子都将被人拿来嘲笑。

就在大家开心调侃丹尼时，阿瑞斯低声问我：“要不要去海边散步？”

天啊，我爱他的声音。

“只要你答应乖乖的。”我抬头看他，眼中含着笑。

“我不能答应不能做到的事情。”他的嘴角浮现狡诈的笑容。 “好吧，我承诺不做你不想要的事情。”

我微微一瞥他，“计划坏了，你以前用过这招，我不会上当。”

他用挫败的表情嘟囔着，“没想你会记住。”

“我记得一切，希腊男神。”

他捏着自己的额头，“这很明显，谁会忘记那天早上的美妙过程呢？你叫得那么——”

“闭嘴！”我立即打断他，“我说的不是那个，我是说情感上，你真傻。”

“你怎么定义傻？”他满脸好奇。

我给了他一个无奈的白眼，“你怎么能在我们第一次亲密后就给我手机？难道不懂得适可而止吗？”

他的表情变得认真，“对不起，我为曾经的所有事情道歉，我没有借口。”他再次伸出双手，“谢谢你没有放弃我，有了你我变得更好。”

我假装很坚强，没有握他手。忽然，他指着沙滩一侧，“范海蟹！”

“啊！哪里？”我本能地靠近他。

他往我这边靠，“来吧，我来保护你。”

我意识到他是在开玩笑，故意去碰他，“去你的。”

阿瑞斯突然单膝跪地，背对着我，“快，爬上来。”

那夏天他让我觉得安全的记忆浮现，晚上我被炸弹攻击的时候，他是那么温柔。“不，我不会走，这次。”

我跨上他的背，他将我举起，让我双腿环绕他的腰，手臂环绕在他脖子上保住平衡。

阿瑞斯在海边走着，我意识到这一天充满了完美的时刻。将脸靠在他的肩膀，感受着海浪的声音和他的体温交融在一起，心中满是惆怅。“如果没有你，我该怎么活？”

我抛开那个念头，低头看着他侧脸，喃喃道：“我爱你。” 

一瞬间静默，他的神情让我忍不住咬咬后唇，直到他开口：“我会留下。”

“你知道，如果你让我留在这里，我会答应。”我心中涌动着复杂的情感。

“但你不会让我这么做。”

他轻叹一声，沉默片刻。

我心里明白，我绝对不能要求他放弃自己的梦想，只为了我。也不能去剥夺他追求自我的权利，这样太自私了。毕竟当我去实现自己的梦想，进入梦寐以求的大学时，不能希望他为我放弃自己的理想。

我从来都相信“爱是无私的”并不是空话，学会把对方的幸福放在心底，是爱最深刻的体现。

我再次将头靠在他的肩膀上，听到他轻声低语，“我也爱你，小巫女。” 

他的话语如春风般温暖，让我感到安心。我忍不住微笑，任由他把我背着，在这片沙滩上细细品味着每一秒的美好。
