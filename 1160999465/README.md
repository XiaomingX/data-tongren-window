# 透过我的窗

“你们见过那个女孩吗？简直太美了！”在教室里，李晓兴奋地对她的朋友们说。

“别那么夸张嘛，晓晓，她不过是个普通的同学而已。”小玲调侃道，抛给她一个坏坏的笑容。

“普通？是你没看到她笑的时候，真的是可以让全世界都黯然失色！”李晓撅着嘴，脑海中不断回放着那个女孩清澈的笑容。

“哟，这可是爱上她的节奏啊！”小杰在一旁调侃，赶紧调整了自己的眼镜，像个小大人一样端起了气。

“我才没呢！”李晓的脸红了，快速转移话题，“你们听说了吗？2月份Netflix要上线一部超级期待的剧，简直就是我的心头好啊！”

“哇，终于可以在网上看了！”小玲的眼睛亮了，意识到李晓是在说的那部剧，“《透过我的窗》吗？我在 instagram 上关注了ATDMVNETFLIX!”

“对对对，2月4号，大家一定要记得看哟！”李晓又开始乐滋滋地讲述她对这部剧的期待，“到时候我会在直播间里回答大家的问题，想一起来吗？”

“当然，别忘了提醒我哦，不然我又要错过好剧了！”小杰摆出一副认真学习的样子，像是在替未来的自己找借口。

“好啦，记得到时候给我发消息，咱们一起看！”李晓微微一笑，“今天就跟我一起陪我等六点钟吧？我们一起刷问答！”

“没问题，随时待命，准备被你激励！”小玲立刻回了个飞吻。

“哈哈，你们太搞笑了！”李晓轻松地笑着，心里却有些暗暗期待着那个女孩能在这新的一年里，也像剧中的角色一样出现在自己的生活里。

李晓看着窗外，开始梦想着那个美丽的冬日，和再也不可思议的新故事。
